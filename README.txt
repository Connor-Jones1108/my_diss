my_flask Web App
================

21105474

Prerequisites:
--------------
1. Python (3.x recommended) installed on your machine.
2. Git Bash or a terminal of your choice.


Setup and Launch:
-----------------
1. Clone the repository:
git clone https://github.com/your-username/your-flask-web-app.git



2. Navigate to the project directory:
cd /c/my_flask/



3. Create a virtual environment (optional but recommended):
python -m venv venv



4. Activate the virtual environment:
- On Windows (Git Bash):
  ```
  source venv/Scripts/activate
  ```
- On macOS or Linux:
  ```
  source venv/bin/activate
  ```

5. Install the required packages:
pip install -r requirements.txt



6. Set the Flask app:
- On Windows (Git Bash):
  ```
  export FLASK_APP=app.py
  ```
- On macOS or Linux:
  ```
  export FLASK_APP=app.py
  ```

7. Set the environment to development mode (optional but recommended for debugging):
- On Windows (Git Bash):
  ```
  export FLASK_ENV=development
  ```
- On macOS or Linux:
  ```
  export FLASK_ENV=development
  ```

8. Run the app:
flask run



9. Open a web browser and navigate to:
http://127.0.0.1:5000/



10. You should see the web app running! Feel free to explore and interact with it.
